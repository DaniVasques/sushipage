# 🍣 JP Sushi Landing Page

Landing page responsiva inspirada em um restaurante japonês, desenvolvida com HTML5, CSS3, JavaScript e GSAP.

---

## 🚀 Tecnologias

- HTML5
- CSS3
- JavaScript
- GSAP

---

## ✨ Funcionalidades

- Layout responsivo
- Animações com GSAP
- Design moderno
- Navegação suave
- Experiência interativa

---

## 👩‍💻 Sobre o Projeto

Este projeto foi desenvolvido durante meus estudos de Front-End na Coti Informática com o objetivo de praticar animações utilizando GSAP, responsividade e criação de interfaces modernas.
